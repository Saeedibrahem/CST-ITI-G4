# MobileXpress - متجر الهواتف الذكية

## نظرة عامة
MobileXpress هو متجر إلكتروني متخصص في بيع الهواتف الذكية والأجهزة المحمولة. يوفر الموقع واجهة مستخدم حديثة وسهلة الاستخدام مع نظام إدارة شامل للمنتجات والمستخدمين.

## الميزات الرئيسية

### 🛍️ للمستخدمين
- **تصفح المنتجات**: عرض جميع المنتجات مع إمكانية التصفية والبحث
- **نظام التسجيل**: تسجيل حساب جديد أو تسجيل الدخول
- **سلة التسوق**: إضافة المنتجات وإدارة المشتريات
- **تفاصيل المنتج**: عرض تفاصيل شاملة لكل منتج
- **تصنيفات متعددة**: تصفح حسب العلامة التجارية والسعر والمواصفات

### 👨‍💼 للبائعين
- **لوحة تحكم البائع**: إحصائيات المبيعات والأداء
- **إدارة المنتجات**: إضافة وتعديل وحذف المنتجات
- **متابعة الطلبات**: عرض وإدارة الطلبات الواردة

### 🔧 للمديرين
- **لوحة تحكم شاملة**: إدارة المستخدمين والمنتجات والطلبات
- **نظام المراجعة**: مراجعة المنتجات قبل النشر
- **إحصائيات متقدمة**: رسوم بيانية وتحليلات مفصلة

## هيكل المشروع

```
CST-ITI-G4 - ai/
├── assets/                 # الملفات الثابتة
│   ├── home_page_img/     # صور الصفحة الرئيسية
│   ├── img/               # الصور العامة
│   └── lib/               # مكتبات خارجية
├── data/                  # ملفات البيانات
│   ├── products.json      # بيانات المنتجات
│   ├── users.json         # بيانات المستخدمين
│   └── admin-demo.json    # بيانات تجريبية للمدير
├── pages/                 # صفحات المشروع
│   ├── login/            # صفحة تسجيل الدخول
│   ├── signup/           # صفحة التسجيل
│   ├── products/         # صفحات المنتجات
│   ├── seller/           # صفحات البائع
│   └── admin.html        # لوحة تحكم المدير
├── src/                  # الكود المصدري
│   ├── css/              # ملفات التنسيق
│   └── js/               # ملفات JavaScript
└── index.html            # الصفحة الرئيسية
```

## كيفية التشغيل

### المتطلبات
- متصفح ويب حديث (Chrome, Firefox, Safari, Edge)
- خادم ويب محلي (مثل Live Server في VS Code)

### خطوات التشغيل
1. **تحميل المشروع**
   ```bash
   git clone [رابط المشروع]
   cd CST-ITI-G4-ai
   ```

2. **تشغيل الخادم المحلي**
   - استخدم Live Server في VS Code
   - أو استخدم أي خادم ويب محلي آخر

3. **فتح المشروع**
   - افتح `index.html` في المتصفح
   - أو استخدم الخادم المحلي

## الصفحات الرئيسية

### 🏠 الصفحة الرئيسية (`index.html`)
- عرض المنتجات المميزة
- التصنيفات الرئيسية
- روابط سريعة للصفحات الأخرى

### 📱 كتالوج المنتجات (`pages/products/index.html`)
- عرض جميع المنتجات
- نظام التصفية والبحث
- ترتيب حسب السعر والمواصفات

### 🔐 تسجيل الدخول (`pages/login/index.html`)
- تسجيل الدخول للمستخدمين الحاليين
- التحقق من البيانات
- إعادة التوجيه حسب نوع المستخدم

### ✍️ التسجيل (`pages/signup/index.html`)
- إنشاء حساب جديد
- اختيار نوع الحساب (عميل/بائع)
- التحقق من صحة البيانات

### 👨‍💼 لوحة تحكم البائع (`pages/seller/`)
- إحصائيات المبيعات
- إدارة المنتجات
- متابعة الطلبات

### 🔧 لوحة تحكم المدير (`pages/admin.html`)
- إدارة شاملة للنظام
- مراجعة المنتجات
- إحصائيات متقدمة

## التقنيات المستخدمة

### Frontend
- **HTML5**: هيكل الصفحات
- **CSS3**: التنسيق والتصميم
- **JavaScript (ES6+)**: التفاعل والوظائف
- **Bootstrap 5**: إطار العمل للتصميم
- **Swiper.js**: عرض الشرائح
- **Chart.js**: الرسوم البيانية

### المكتبات
- **Font Awesome**: الأيقونات
- **CryptoJS**: تشفير البيانات
- **Google Fonts**: الخطوط
